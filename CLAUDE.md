# شاشة مواقيت الصلاة للمساجد — دليل المشروع

## ⚠️ الهوية والفصل
هذا **مشروع شاشة المسجد** (تلفزيون/شاشة عرض + لوحة إدارة مركزية + PWA).
**تطبيق الهاتف الشخصي** منفصل تماماً في `..\تطبيق مواقيت الصلاة` (مستودع prayer-times-app) — **لا تخلط بينهما أبداً** (طلب صريح من حسن).
محرك `js/praytimes.js` منسوخ في المشروعين — أي إصلاح فيه يُطبَّق في المكانين.

## المستودع والنشر
- GitHub: `hasanawida/mosque-screen` — الموقع الحي: https://hasanawida.github.io/mosque-screen/
- أي push إلى main يصل لكل الشاشات المتصلة تلقائياً (SW network-first + فحص كل ساعة + إعادة تحميل يومية 00:20)
- **مع كل إصدار**: حدّث `APP_VERSION` في js/version.js + `VERSION` في sw.js (vN) + CHANGELOG.md ثم tag
- CI: اختبارات + فحص صياغة على كل push (يجب أن تبقى خضراء)
- غلاف TWA للشاشة (PWABuilder): package `io.github.hasanawida.twa`؛ ملفات التوقيع في `Documents\مفاتيح-تطبيق-مواقيت-الصلاة` (لا تُرفع للمستودع أبداً)؛ assetlinks.json في مستودع `hasanawida.github.io`

## البنية
- `index.html` الشاشة | `admin.html` لوحة الإدارة | `diagnostics.html` فحص النظام
- `js/app.js` منطق الشاشة (الحالات: أذان ← انتظار إقامة ← صلاة ← أذكار) — **العرض بتوقيت مدينة المسجد** (cityParts/tzOffsetHours) وليس توقيت الجهاز
- `js/settings.js` الإعدادات + `sanitizeSettings` (تنقية صارمة لكل إعدادات واردة: تحديث بعيد/استيراد/معاينة)
- `js/praytimes.js` المحرك الفلكي (12 اختباراً في tests/) | `js/cities.js` 42 مدينة مع IANA tz
- `js/storage.js` IndexedDB للوسائط (صورة الجامع، أذان مرفوع، خلفيات)
- `audio/` 4 أذانات مدمجة | `sw.js` أوفلاين + تحديث ذاتي
- الإدارة عن بُعد: لوحة admin تنشر إعدادات JSON لكل مسجد إلى GitHub Gist، والشاشات تسحبها دورياً (رمز gist في تخزين منفصل، خيار جلسة فقط)

## قواعد مهمة تعلمناها
- الأذان له أولوية مطلقة في currentEvent (يُفحص لكل الصلوات قبل باقي النوافذ)
- لا يُقاطَع صوت أذان يعمل عند حفظ الإعدادات (unlockAudio يتخطى عنصراً يعمل)
- وقت الجمعة الثابت: isNaN وليس `||` (الصفر falsy!)
- صوت الأذان لا يعمل من صفحة مخفية (document.hidden) — مقصود
- فحص الاتصال عبر HEAD لموقعنا (يتجاوز SW لأن معالج fetch يتجاهل غير GET)

## أوامر

- فحص محلي: `python -m http.server 8741` من مجلد المشروع (المعاينة لا تعيد تحميل file:// — استخدم http)
