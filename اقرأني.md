# 🕌 شاشة مواقيت الصلاة

شاشة عرض احترافية للمساجد — تعمل بدون إنترنت على أي جهاز فيه متصفح.

**تطوير: حسن عويضة (Hasan Awida)** · [github.com/hasanawida](https://github.com/hasanawida)

## التشغيل

1. افتح الملف `index.html` بمتصفح كروم أو إيدج.
2. اضغط **F11** للعرض بملء الشاشة.
3. اضغط حرف **S** (أو زر الترس أعلى اليسار) لفتح الإعدادات.

### للتشغيل الدائم على شاشة المسجد (مع صوت تلقائي)

أنشئ اختصاراً لكروم بهذه الإضافات:

```
chrome.exe --kiosk --autoplay-policy=no-user-gesture-required "C:\مسار\المجلد\index.html"
```

بهذا الوضع تفتح الشاشة كاملة تلقائياً ويعمل صوت الأذان بدون أي ضغطة.

## الميزات

- حساب فلكي محلي للأوقات (بدون إنترنت) — 6 طرق حساب معتمدة
- 42 مدينة جاهزة + إمكانية أي إحداثيات
- التاريخ الهجري (أم القرى) مع إمكانية التعديل ± يومين
- شاشة أذان تلقائية + صوت أذان: **الحرم المكي، المسجد النبوي، المسجد الأقصى** (مدمجة تعمل بدون إنترنت — المصدر archive.org)، أو تنبيه مدمج، أو ملف ترفعه بنفسك
- تكرار الأذكار حسب السنة: اكتب «الذكر | العدد» (مثل: سُبْحَانَ اللهِ | 33) فيظهر عدّاد مباشر على الشاشة وتطول مدة العرض تلقائياً بما يكفي للترديد
- عد تنازلي للإقامة + شاشة "أقيمت الصلاة — أغلقوا الهواتف"
- شاشة أذكار وأدعية ما بعد الصلاة — مدة عرض كل ذكر قابلة للضبط (افتراضياً 30 ثانية لكل ذكر، 8 دقائق إجمالاً)
- شاشة إعلانات كاملة بجدولة يحددها المدير: كل X دقيقة و/أو بأوقات محددة (مثل 21:30) — لا تظهر أبداً أثناء الأذان أو الصلاة
- خلفيات صور (مساجد أو غيرها): ارفع حتى 12 صورة، تتبدل تلقائياً مع تعتيم قابل للضبط لوضوح النص
- يوم الجمعة تلقائي: خطبة ثم إقامة
- صورة/شعار الجامع (تُرفع من الإعدادات وتُحفظ على الجهاز)
- إعلانات المسجد في الشريط السفلي مع شارة «إعلان»
- مؤشر اتصال بالإنترنت (متصل/غير متصل)
- 5 تصاميم جاهزة + تصميم بألوان مخصصة بالكامل
- اتجاه العرض من الإعدادات: تلقائي / أفقي دائماً / عمودي دائماً — مع تصميم مخصص لكل اتجاه
- **مناسبات هجرية تلقائية:** عاشوراء، المولد النبوي، الإسراء والمعراج، الأعياد، الأيام البيض، سورة الكهف يوم الجمعة، تذكير صيام الاثنين والخميس — تظهر بيومها تلقائياً (واليوم الهجري يتقدم شرعياً بعد المغرب)
- **وضع رمضان تلقائي:** شريط «رمضان مبارك» مع الإمساك والإفطار، العد التنازلي يصبح «المتبقي للإفطار 🌙»، «يليها التراويح» على العشاء، وتذكير ليلة القدر في العشر الأواخر
- PWA كامل: يثبّت كتطبيق، يعمل بدون إنترنت، ويحدّث نفسه من الاستضافة

## 🔒 ملاحظات أمان مهمة

- **رمز GitHub:** استخدم دائماً Fine-grained token بصلاحية **Gists فقط** (الخطوات داخل نافذة «إعداد النشر») — حتى لو تسرب لا يمس حسابك. على جهاز مشترك ألغِ «تذكّر الرمز» فيُمسح عند إغلاق المتصفح.
- **ملف الإعدادات البعيد (Gist):** أي شخص يملك الرابط يستطيع قراءته — لا تضع فيه أرقام هواتف أو بيانات متبرعين أو أي معلومات خاصة. الإعلانات وأوقات الإقامة والألوان آمنة تماماً.
- الشاشة **تنقّي تلقائياً** أي إعدادات واردة عن بُعد (أنواع وحدود صارمة) — ملف تالف لن يكسر العرض.

## 🩺 صفحة فحص النظام

افتح `diagnostics.html` (أو من رابط داخل نافذة الإعدادات) لفحص فوري: تطابق المنطقة الزمنية، مواقيت اليوم، حالة الاتصال، التخزين والوسائط، الصوت، آخر مزامنة، وإصدار التطبيق. مفيدة جداً عند تركيب شاشة جديدة أو تشخيص مشكلة عن بُعد.

## 🎛 لوحة الإدارة المركزية (`admin.html`)

لإدارة **كل الجوامع وكل الشاشات من مكان واحد** — افتح `admin.html`:

- أضف جامعاً لكل شاشة (زر «＋ إضافة»)، وكرّر أي شاشة موجودة بضغطة
- عدّل كل إعدادات الشاشة: الإعلانات، أوقات الإقامة، الألوان، المدينة...
- **معاينة حية**: شاهد الشاشة بشكلها النهائي قبل النشر
- **☁ نشر الكل**: يرفع إعدادات كل الشاشات دفعة واحدة عبر GitHub Gist (مجاني، بدون سيرفر)

### ربط شاشة جديدة باللوحة (مرة واحدة فقط)

1. في اللوحة: «إعداد النشر» ← ضع رمز GitHub (بصلاحية gist فقط) ← «نشر الكل»
2. اختر الجامع واضغط «📋 نسخ رابط الشاشة»
3. على شاشة المسجد: الإعدادات (S) ← «التحديث عن بُعد» ← الصق الرابط وفعّله

من الآن، أي تعديل تنشره من اللوحة يصل لكل الشاشات المتصلة تلقائياً خلال دقائق — من أي مكان في العالم.

## 🚀 PWA — التحديث الكامل عن بُعد (موصى به)

عند استضافة المشروع على رابط، كل شاشة تفتح الرابط تصبح **تطبيقاً كاملاً يعمل بدون إنترنت ويحدّث نفسه تلقائياً** — أي تعديل ترفعه على الاستضافة يصل لكل الشاشات خلال ساعة كحد أقصى (بما فيها الكود نفسه، مش بس الإعدادات).

### الاستضافة المجانية على GitHub Pages (مرة واحدة)

1. أنشئ مستودعاً جديداً على GitHub (مثلاً باسم `mosque-screen`)
2. ارفع كل ملفات هذا المجلد إليه
3. من إعدادات المستودع ← Pages ← فعّل النشر من فرع main
4. رابطك يصبح: `https://اسمك.github.io/mosque-screen/`
5. افتح الرابط على أي شاشة — من أول زيارة تُخزَّن كل الملفات (حتى الأذانات) وتشتغل لاحقاً بدون إنترنت

**للتحديث لاحقاً:** عدّل الملفات وارفعها + غيّر رقم النسخة في أول سطر من `sw.js` (مثلاً `v1` ← `v2`) — كل الشاشات المتصلة تتحدث تلقائياً.

## 🖥 الجهاز المخصص لتشغيل الشاشة

مش كل تلفزيون بيدعم PWA — الحل جهاز صغير يوصل بأي شاشة عبر HDMI:

### الخيار الأفضل: أندرويد بوكس (⁓100-200 شيكل)
أي جهاز Android TV Box من أي محل إلكترونيات:
1. ثبّت تطبيق **Fully Kiosk Browser** من متجر Google Play (الأفضل لهذا الغرض)
2. ضع رابط الشاشة (أو افتح الملفات المحلية)
3. من إعدادات التطبيق فعّل: Launch on Boot (إقلاع تلقائي) + Autoplay Media (صوت تلقائي) + Keep Screen On
4. النتيجة: الشاشة بتشتغل لحالها مع كل تشغيل للكهرباء، بملء الشاشة، مع الصوت

### بدائل ممتازة:
- **كمبيوتر صغير / لابتوب قديم (ويندوز):** اختصار كروم في مجلد بدء التشغيل:
  ```
  chrome.exe --kiosk --autoplay-policy=no-user-gesture-required "الرابط أو مسار index.html"
  ```
- **Raspberry Pi (⁓250 شيكل):** نظام مجاني + Chromium بوضع kiosk — الأوفر للتشغيل الدائم (استهلاك كهرباء ضئيل)
- **شاشة أندرويد ذكية مباشرة:** إذا فيها متصفح كروم، افتح الرابط وثبّته كتطبيق (قائمة كروم ← إضافة إلى الشاشة الرئيسية)

## التحديث عن بُعد

تقدر تغيّر إعدادات الشاشة (الإعلانات، أوقات الإقامة...) من أي مكان بدون ما تروح للمسجد:

1. من الإعدادات اضغط **تصدير الإعدادات** — بتحصل على ملف JSON.
2. عدّل فيه ما تريد وارفعه على الإنترنت (الأسهل: [GitHub Gist](https://gist.github.com) ثم زر Raw).
3. ضع الرابط في الإعدادات ← «الاتصال والتحديث عن بُعد» وفعّله.
4. الشاشة ستفحص الرابط دورياً وتطبّق أي تغيير تلقائياً وهي متصلة.

ملاحظة: الملف البعيد يكفي أن يحتوي الحقول المطلوب تغييرها فقط، مثلاً:

```json
{
  "announcements": ["درس فقه بعد العشاء يوم الثلاثاء"],
  "iqama": { "fajr": 25 }
}
```
