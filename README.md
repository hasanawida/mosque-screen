## ✨ الميزات

- **حساب فلكي محلي** للمواقيت (بدون أي API) — 6 طرق حساب معتمدة، مذهبا العصر، معالجة خطوط العرض العالية، مثبَّت بـ11 اختباراً آلياً
- **يعمل بدون إنترنت بالكامل** — PWA يخزّن كل شيء من أول زيارة ويحدّث نفسه تلقائياً عند توفر الاتصال
- **دقة المنطقة الزمنية**: كل مدينة بمنطقتها الرسمية (IANA) — الأوقات تُعرض بتوقيت المسجد حتى لو كان الجهاز مضبوطاً خطأً، مع تحذير واضح
- **الدورة الكاملة تلقائياً**: شاشة أذان (بصوت الحرم المكي / النبوي / الأقصى / عبد الرحمن الهندي) ← عد تنازلي للإقامة ← شاشة «أقيمت الصلاة — أغلقوا الهواتف» ← أذكار ما بعد الصلاة **بعدّاد تكرار حسب السنة** (×3، ×33...)
- **الجمعة ورمضان والمناسبات**: خطبة ثم إقامة، شريط رمضان مع الإمساك والإفطار وعد تنازلي للإفطار، وعاشوراء والأعياد والأيام البيض وسورة الكهف تلقائياً بيومها (باليوم الهجري الشرعي الذي يبدأ من المغرب)
- **إعلانات المسجد**: في الشريط السفلي و/أو شاشة كاملة بجدولة يحددها الإمام — لا تظهر أبداً وقت الصلاة
- **تخصيص كامل**: 5 ثيمات + ألوان مخصصة، صورة الجامع، خلفيات متبدلة، أفقي/عمودي، 12/24 ساعة، 42 مدينة جاهزة أو أي إحداثيات
- **لوحة إدارة مركزية**: أدر عشرات المساجد من مكان واحد وانشر التحديثات عن بُعد (عبر GitHub Gist — بدون خادم)

## 🚀 التشغيل السريع

**على أي جهاز:** افتح الرابط أعلاه ← F11 للملء الكامل ← حرف S للإعدادات.

**📱 كتطبيق هاتف:** افتح الرابط في كروم ← قائمة (⋮) ← «تثبيت التطبيق» — لوحة الإدارة والشاشة يتثبتان كتطبيقين مستقلين يعملان بدون إنترنت. ولملف APK: [pwabuilder.com](https://www.pwabuilder.com) بضغطة واحدة.

**على شاشة المسجد (أندرويد بوكس):** راجع [دليل-اندرويد-بوكس.md](دليل-اندرويد-بوكس.md) — تثبيت Fully Kiosk مع إقلاع تلقائي وصوت بدون لمس.

**التوثيق الكامل:** [اقرأني.md](اقرأني.md)
